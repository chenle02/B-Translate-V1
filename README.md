# B-Translate V1

Global translator Android app.

Features:
- Auto-detect source language
- Large target language list
- Real online translation attempt
- Google Translate fallback
- Swap languages
- Copy result
- Translation history
- Favorite translations
- B-Map and B-Notes shortcuts

Note: Internet is required for real translation. Direct translation uses an online translate endpoint; if it fails, the app opens Google Translate as fallback.
